Getting Started

This library displays music with beginner and learning friendly features.

These features include:

  • Display the note name while a note is hovered over
  • Click on notes to display their note names
  • Click on notes to play their sounds
  • Buttons to show and hide all note names
  • A beginner reading music quiz

In order to setup this library the following script needs to be included in the html:
<script defer type="text/javascript" src='./js/MusicDisplay.js'><script>

No other external libraries/modules or css files are needed.
